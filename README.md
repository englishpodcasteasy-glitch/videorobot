# VideoRobot 🤖🎬

یک پلتفرم کامل برای تولید ویدیوهای خودکار با قابلیت‌های:
- رونویسی هوشمند صوت با Whisper
- تولید زیرنویس انیمیشنی

- رندر ویدیو با FFmpeg
